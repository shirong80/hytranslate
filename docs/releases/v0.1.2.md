v0.1.2 🛠️ 전체화면 팝업 표시 수정 / Full-Screen Popup Fix

> 어떤 화면에서든 번역. 데이터는 기기를 벗어나지 않습니다.
> Translate on any screen. Your data never leaves your device.

## 새로운 기능

1. **전체화면 위 번역 팝업 표시 수정**: 다른 앱이 네이티브 전체화면일 때 `Cmd+Shift+T` 번역 팝업이 그 위에 뜨지 않고 일반 데스크탑으로 떨어지던 문제를 수정했습니다. 이제 팝업이 현재 전체화면 Space 위에 바로 표시되고, 전체화면을 벗어나지 않은 채 입력 포커스를 받아 바로 타이핑할 수 있습니다.

## Changelog

1. **Full-screen translation popup fix**: Fixed the `Cmd+Shift+T` translation popup not showing on top of another app's native full-screen Space (it previously fell back to the regular desktop). The popup now appears directly on the current full-screen Space and takes input focus without switching you out of full screen.

## 시작하기 / Getting Started

1. **Ollama 설치 / Install Ollama**: https://ollama.com/download
2. **모델 받기 / Pull a model**:
   - 12GB+ RAM: `ollama pull hf.co/tencent/Hy-MT2-7B-GGUF:Q4_K_M`
   - 8GB RAM: `ollama pull hf.co/tencent/Hy-MT2-1.8B-GGUF:Q4_K_M`
3. **어디서나 번역 / Translate anywhere**: `Cmd+Shift+T` (전체화면 앱 위에서도 동작 / works over full-screen apps too)

## 개인정보 / Privacy

번역은 localhost를 벗어나지 않습니다. 텔레메트리도, 클라우드도 없습니다. / Translation never leaves localhost. No telemetry, no cloud.

## 시스템 요구사항 / System Requirements

macOS 13 Ventura+ · Apple Silicon 권장 (Intel 지원) / macOS 13 Ventura+, Apple Silicon recommended (Intel supported).
