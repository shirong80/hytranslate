v0.1.1 🖥️ 전체화면에서도 번역 / Translate in Full Screen

> 어떤 화면에서든 번역. 데이터는 기기를 벗어나지 않습니다.
> Translate on any screen. Your data never leaves your device.

## 새로운 기능

1. **전체화면 앱 위에 번역 팝업 표시**: 다른 앱이 전체화면(full-screen Space) 상태일 때도 `Cmd+Shift+T`로 번역 팝업을 바로 그 위에 띄울 수 있습니다. 팝업은 메뉴 막대 아래·전체화면 콘텐츠 위 레벨에 표시되며, 현재 Space를 벗어나지 않습니다.
2. **멀티 디스플레이 전체화면 팝업 위치 보정**: 여러 디스플레이를 사용할 때 커서가 있는 화면(전체화면 포함)에 팝업이 정확히 중앙 배치됩니다. 이전에 팝업이 엉뚱한 Space에 떠서 보이지 않던 문제를 수정했습니다.

## Changelog

1. **Translation popup over full-screen apps**: Press `Cmd+Shift+T` to bring up the translation popup on top of another app's full-screen Space. The popup sits below the menu bar but above full-screen content, and it does not switch you out of the current Space.
2. **Multi-display full-screen popup placement fix**: With multiple displays, the popup is now centered correctly on the screen where your cursor is, including a full-screen one. Fixes a case where the popup could appear on the wrong Space and stay invisible.

## 시작하기 / Getting Started

1. **Ollama 설치 / Install Ollama**: https://ollama.com/download
2. **모델 받기 / Pull a model**:
   - 12GB+ RAM: `ollama pull hf.co/tencent/Hy-MT2-7B-GGUF:Q4_K_M`
   - 8GB RAM: `ollama pull hf.co/tencent/Hy-MT2-1.8B-GGUF:Q4_K_M`
3. **어디서나 번역 / Translate anywhere**: `Cmd+Shift+T` (전체화면 앱 위에서도 동작 / works over full-screen apps too)

## 개인정보 / Privacy

번역은 localhost를 벗어나지 않습니다. 텔레메트리도, 클라우드도 없습니다. / Translation never leaves localhost. No telemetry, no cloud.
