v0.2.0 ✋ 직접 저장하고 옮기기 / Save & Move, On Your Terms

> 어떤 화면에서든 번역. 데이터는 기기를 벗어나지 않습니다.
> Translate on any screen. Your data never leaves your device.

## 새로운 기능

1. **Cmd+Enter 수동 저장**: 번역이 끝나면 `Cmd+Enter` 로 그 결과를 이력에 저장하고 입력을 비워 바로 다음 번역으로 넘어갑니다. 이전에는 모든 번역이 자동으로 저장됐지만, 이제 남기고 싶은 번역만 직접 골라 저장합니다. (이력 저장을 꺼두면 저장되지 않습니다.)
2. **팝업 창 드래그 이동**: 번역 팝업의 헤더를 잡고 끌어 원하는 위치로 옮길 수 있습니다.
3. **결과가 길어져도 팝업 위치 고정**: 번역 결과가 길어져 팝업 높이가 늘어날 때도 창이 제자리에 머물러, 옮겨 둔 위치가 흔들리지 않습니다.

## Changelog

1. **Manual save with Cmd+Enter**: After a translation completes, press `Cmd+Enter` to save it to history and clear the input for the next one. Translations are no longer saved automatically — you now choose which ones to keep. (Nothing is saved if history saving is turned off.)
2. **Draggable popup window**: Grab the translation popup's header and drag it to wherever you want it.
3. **Stable popup position on longer results**: When a longer translation grows the popup's height, the window stays put — the spot you moved it to no longer shifts.

## 시작하기 / Getting Started

1. **Ollama 설치 / Install Ollama**: https://ollama.com/download
2. **모델 받기 / Pull a model**:
   - 12GB+ RAM: `ollama pull hf.co/tencent/Hy-MT2-7B-GGUF:Q4_K_M`
   - 8GB RAM: `ollama pull hf.co/tencent/Hy-MT2-1.8B-GGUF:Q4_K_M`
3. **어디서나 번역 / Translate anywhere**: `Cmd+Shift+T` (전체화면 앱 위에서도 동작 / works over full-screen apps too)

## 개인정보 / Privacy

번역은 localhost를 벗어나지 않습니다. 텔레메트리도, 클라우드도 없습니다. / Translation never leaves localhost. No telemetry, no cloud.
