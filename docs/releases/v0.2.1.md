v0.2.1 🔧 듀얼 모니터 팝업 위치 수정 / Multi-Monitor Popup Placement Fix

> 어떤 화면에서든 번역. 데이터는 기기를 벗어나지 않습니다.
> Translate on any screen. Your data never leaves your device.

## 새로운 기능

1. **커서가 있는 화면에 팝업 표시**: `Cmd+Shift+T` 로 팝업을 열 때, 해상도나 배율이 서로 다른 모니터를 함께 쓰더라도 마우스 커서가 있는 화면 한가운데에 정확히 표시됩니다. 이전에는 모니터마다 배율이 다르면 팝업이 엉뚱한 화면이나 잘못된 위치에 뜰 수 있었습니다.
2. **팝업 높이 복원 안정화**: 팝업을 닫았다 다시 열 때 번역 결과 길이에 맞춰 높이를 다시 계산해 적용합니다. 빠르게 닫고 여는 과정에서 늦게 끝난 크기 변경이 최신 높이를 덮어쓰던 문제를 없앴습니다.

## Changelog

1. **Popup opens on the monitor under the cursor**: When you open the popup with `Cmd+Shift+T`, it now appears centered on the screen where your mouse cursor is — even across monitors with different resolutions or scaling. Previously, mismatched display scales could place the popup on the wrong screen or at the wrong position.
2. **Reliable popup height restore**: Reopening the popup recomputes and applies its height for the current translation length. A late-finishing resize can no longer overwrite the latest height when closing and reopening quickly.

## 시작하기 / Getting Started

1. **Ollama 설치 / Install Ollama**: https://ollama.com/download
2. **모델 받기 / Pull a model**:
   - 12GB+ RAM: `ollama pull hf.co/tencent/Hy-MT2-7B-GGUF:Q4_K_M`
   - 8GB RAM: `ollama pull hf.co/tencent/Hy-MT2-1.8B-GGUF:Q4_K_M`
3. **어디서나 번역 / Translate anywhere**: `Cmd+Shift+T` (전체화면 앱 위에서도 동작 / works over full-screen apps too)

## 개인정보 / Privacy
