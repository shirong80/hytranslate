v0.1.0 🌅 첫 출시 / First Light

로컬 Ollama와 Tencent Hy-MT2로 한국어·중국어를 영어로, 전부 내 Mac 안에서 번역하는 macOS 메뉴바 앱.
A macOS menubar app that translates Korean and Chinese into English entirely on-device via Ollama and Tencent Hy-MT2.

## 새로운 기능

1. **실시간 스트리밍 번역**: 한국어·간체·번체 중국어를 로컬 Ollama 모델로 영어로 번역하며, 결과가 생성되는 즉시 화면에 점진적으로 나타납니다.
2. **입력 중 자동 재번역**: 입력을 멈추면 500ms 뒤 자동으로 번역되고, 계속 수정하면 직전 요청은 취소됩니다. `Cmd+Enter`로 지연 없이 바로 번역하거나 재번역할 수 있습니다.
3. **언어 자동 감지와 수동 선택**: 입력 언어를 Auto로 두면 한국어·간체·번체를 자동 감지해 배지로 표시하고, 드롭다운에서 직접 언어를 지정할 수도 있습니다.
4. **전역 단축키로 플로팅 팝업 호출**: 어떤 앱에서든 `Cmd+Shift+T`를 눌러 화면 중앙의 번역 팝업을 띄울 수 있습니다. 팝업에서 `Esc`로 닫고, `Cmd+Enter`로 즉시 번역, `Cmd+C`로 결과를 복사합니다.
5. **메뉴바 트레이 팝오버**: 메뉴바 아이콘을 좌클릭하면 번역과 최근 5건을 보여주는 작은 팝오버가 열리고, 포커스를 잃으면 자동으로 닫힙니다. 우클릭하면 메인 창 열기, 이력 보기, 설정, 종료 메뉴가 나타납니다.
6. **클립보드에서 번역**: 팝업이나 메뉴바 팝오버의 붙여넣기 버튼으로 macOS 클립보드의 텍스트를 입력란에 가져옵니다. 클립보드가 비어 있거나 이미지처럼 지원하지 않는 형식이면 인라인 안내가 표시됩니다.
7. **클립보드로 복사**: 영어 결과를 한 번에 복사할 수 있고, 짧은 확인 메시지와 함께 실패 시 오류 안내를 보여줍니다. 설정에서 번역 완료 후 자동 복사를 켤 수도 있습니다.
8. **번역 이력 자동 저장**: 이력 저장이 켜져 있으면 완료된 번역의 원문, 번역문, 언어, 모델, 소요 시간이 로컬 SQLite 데이터베이스에 자동 저장됩니다.
9. **전문 검색**: FTS5 전문 색인으로 원문과 번역문을 키워드 검색하고, 관련도 순으로 정렬된 결과를 봅니다.
10. **즐겨찾기·태그·필터**: 상세 화면에서 기록에 별표를 달거나 자유로운 태그를 붙일 수 있고, 즐겨찾기만 보기나 태그로 목록을 걸러낼 수 있습니다.
11. **이력 복사·삭제·내보내기**: 선택한 기록의 원문이나 번역문을 복사하고, 개별 기록 삭제나 전체 삭제(확인 절차 포함)를 할 수 있으며, 전체 이력을 CSV(스프레드시트 수식 주입 방지 적용) 또는 JSON 파일로 내보낼 수 있습니다.
12. **첫 실행 설정 마법사**: 처음 실행하면 7단계 설정 마법사(환영, 환경, Ollama, 모델, 권한, 이력, 완료)가 나타나며, 완료하기 전까지만 표시됩니다.
13. **시스템 환경 점검**: 마법사가 macOS 버전, 프로세서(Apple Silicon 또는 Intel), 총 RAM을 감지해 표시하고, 미지원 macOS·Intel 성능·메모리 부족을 경고합니다.
14. **Ollama 설치·실행 감지**: Ollama 설치 여부와 실행 여부를 점검해 실행 중·설치됨(중지)·미설치 세 가지 상태를 명확히 보여줍니다. 미설치 시 공식 다운로드 페이지(ollama.com/download)를 열고, 중지 상태면 클릭으로 실행한 뒤 백오프 폴링으로 실행을 확인하며, 언제든 다시 점검할 수 있습니다.
15. **RAM 기반 모델 추천과 다운로드**: RAM이 12GB 미만이면 Hy-MT2 1.8B, 그 이상이면 Hy-MT2 7B를 추천 배지와 함께 제안합니다. 두 모델 중 골라 다운로드하면 상태·퍼센트·다운로드/전체 용량을 진행 막대로 실시간 확인할 수 있고, 진행 중 취소도 가능합니다. 이미 설치된 모델은 표시됩니다.
16. **접근성 권한 설정 열기**: 전역 단축키에 필요한 권한을 위해 마법사와 설정에서 macOS 접근성 설정 화면을 바로 열 수 있습니다.
17. **마법사 완료 시 활성 모델 지정**: 마법사를 마치면 선택한 모델이 활성 번역 모델로 저장되어 첫 번역이 곧바로 동작합니다.
18. **글자 수 표시와 입력 제한**: 실시간 글자 수가 보이며 제한 초과 시 빨간색으로 바뀝니다(메인 창 30,000자, 플로팅 팝업과 메뉴바 5,000자).
19. **번역 소요 시간 표시**: 번역이 끝나면 걸린 시간이 모델 이름 옆 배지로 표시됩니다.
20. **라이트·다크·시스템 테마**: 설정에서 라이트, 다크, 시스템 외관을 선택하면 메인 창·팝업·메뉴바에 즉시 적용됩니다(시스템 모드는 macOS를 따릅니다).
21. **설정 영구 저장과 토글**: 설정은 디스크에 저장되어 다음 실행 때 복원되며, 파일이 없거나 손상되면 기본값으로 안전하게 되돌아갑니다. 이력 저장, 자동 복사, 로그인 시 자동 실행, Dock 아이콘 숨기기(메뉴바 전용 모드)를 켜고 끌 수 있습니다.
22. **Ollama 엔드포인트와 단축키 설정**: 로컬 Ollama 서버 주소를 설정할 수 있으며 번역을 내 기기에 묶어두기 위해 루프백이 아닌 주소는 거부됩니다. 전역 단축키도 변경할 수 있고 저장 즉시 다시 등록되며, 잘못된 조합은 거부되고 이전의 정상 단축키로 자동 복원됩니다.
23. **닫아도 종료되지 않는 메인 창**: 메인 창을 닫으면 종료되지 않고 숨겨져, 앱은 메뉴바에 계속 떠 있고 다시 열 수 있습니다.

## Changelog

1. **Real-time streaming translation**: Translate Korean, Simplified, and Traditional Chinese into English with the local Ollama model, watching the output appear progressively as it is generated instead of waiting for the whole result.
2. **Auto re-translation while typing**: The app translates automatically 500ms after you stop typing and cancels the previous in-flight request when you keep editing. Use `Cmd+Enter` to translate or re-translate immediately with no delay.
3. **Automatic detection and manual selection**: Leave the input language on Auto to detect Korean, Simplified, or Traditional Chinese and show it as a badge, or pick the language yourself from the dropdown.
4. **Global hotkey summons a floating popup**: Press `Cmd+Shift+T` from any app to toggle a centered translation popup. In the popup, `Esc` closes, `Cmd+Enter` translates immediately, and `Cmd+C` copies the result.
5. **Menubar tray popover**: Left-click the menubar icon to open a compact popover that translates and shows your 5 most recent translations, auto-hiding when it loses focus. Right-click for a menu to open the main window, jump to history, open settings, or quit.
6. **Translate from clipboard**: Use the paste button in the popup or menubar popover to pull text from the macOS clipboard into the input, with inline notices when the clipboard is empty or holds an unsupported type like an image.
7. **Copy to clipboard**: Copy the English result in one click with a brief confirmation and an error notice if copying fails. You can also enable auto-copy after a translation finishes in Settings.
8. **Automatic translation history**: When history saving is on, completed translations are saved automatically to a local SQLite database, capturing source text, translation, language, model, and duration.
9. **Full-text search**: Search saved translations by keyword across both source and translated text using an FTS5 full-text index, with results ranked by relevance.
10. **Favorites, tags, and filters**: Star a record or add freeform tags from its detail view, and filter the list to favorites only or by a specific tag.
11. **Copy, delete, and export history**: Copy the source or translated text of a selected record, delete an individual record or wipe all history (with a confirmation prompt), and export your full history to a CSV file (with spreadsheet formula-injection neutralization) or a JSON file.
12. **First-run setup wizard**: On first launch a guided 7-step wizard (welcome, environment, Ollama, model, permissions, history, done) walks you through setup and only appears until you complete it.
13. **System environment check**: The wizard detects and displays your macOS version, processor type (Apple Silicon or Intel), and total RAM, warning about unsupported macOS, Intel performance, or low memory.
14. **Ollama install and running detection**: The wizard checks whether Ollama is installed and running, showing one of three clear states: running, installed-but-stopped, or not installed. When it is missing, open the official download page (ollama.com/download); when stopped, launch it with a click and the app polls its status with backoff until it confirms running. Re-check at any time.
15. **RAM-based model recommendation and download**: The wizard recommends Hy-MT2 1.8B under 12 GB of RAM and Hy-MT2 7B otherwise, with a recommended badge. Pick either model, download it while watching live status, percentage, and downloaded/total bytes on a progress bar, and cancel mid-download if needed. Already-installed models are marked.
16. **Open Accessibility settings**: Jump straight to the macOS Accessibility settings pane from the wizard and from Settings to grant the access global shortcuts need.
17. **Active model set on finish**: Completing the wizard saves your chosen model as the active translation model so your first translation works immediately.
18. **Live character count with input limits**: A running character count turns red when you exceed the limit (30,000 in the main window, 5,000 in the floating popup and menubar).
19. **Translation duration display**: After a translation completes, the time it took appears as a badge next to the model name.
20. **Light / Dark / System theme**: Choose Light, Dark, or System appearance in Settings and it applies instantly across the main window, popup, and menubar (System mode follows macOS).
21. **Persistent settings and toggles**: Settings are saved to disk and restored on the next launch, with safe fallback to defaults if the file is missing or corrupt. Toggle history saving, auto-copy, start at login, and hiding the Dock icon (menubar-only mode).
22. **Ollama endpoint and hotkey settings**: Set the local Ollama server address (non-loopback addresses are rejected to keep translation on your machine), and change the global shortcut, which re-registers immediately on save while rejecting invalid combos and restoring the previous working shortcut.
23. **Close-to-hide main window**: Closing the main window hides it instead of quitting, so the app keeps running in the menubar and can be reopened.

## 시작하기 / Getting Started

1. **Ollama 설치 / Install Ollama**: https://ollama.com/download
2. **모델 받기 / Pull a model**:
   - 12GB+ RAM: `ollama pull hf.co/tencent/Hy-MT2-7B-GGUF:Q4_K_M`
   - 8GB RAM: `ollama pull hf.co/tencent/Hy-MT2-1.8B-GGUF:Q4_K_M`
3. **앱 온보딩 / In-app onboarding**: 첫 실행 마법사가 위 과정을 대신 해줍니다. / The first-run wizard does this for you.
4. **어디서나 번역 / Translate anywhere**: `Cmd+Shift+T`

## 개인정보 / Privacy

번역은 localhost를 벗어나지 않습니다. 텔레메트리도, 클라우드도 없습니다. / Translation never leaves localhost. No telemetry, no cloud.

## 시스템 요구사항 / System Requirements

macOS 13 Ventura+ · Apple Silicon 권장 (Intel 지원) / macOS 13 Ventura+, Apple Silicon recommended (Intel supported).
