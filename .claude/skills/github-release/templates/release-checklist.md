## 4. 게시 명령 (확인 요청)

아래 명령으로 `v{버전}` {정식/사전배포/초안} 릴리스를 게시합니다. 진행할까요?
(검토만 원하시면 `--draft`로 만들 수 있습니다.)

```bash
gh release create v{버전} \
  --target {branch} \
  --title "{헤더 1행}" \
  --notes-file {body-file} \
  {--latest | --prerelease | --draft}
```
